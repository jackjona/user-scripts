# User-Scripts
A collection of user scripts made by me.

Greasy Fork Links:
- [UofT - Course Hover Popup](https://greasyfork.org/en/scripts/568651-uoft-course-hover-popup) - A user script to view course details when hovering over course codes on the University of Toronto academic calendar website
- [UofT Library Proxy](https://greasyfork.org/en/scripts/568654-uoft-library-proxy) - A user script that adds a floating button to proxy Nature and ScienceDirect pages through UofT library access
- [CommonSenseMedia Gate Remover + Auto Close](https://greasyfork.org/en/scripts/568652-commonsensemedia-gate-remover-auto-close) - A user script that removes gate-wrapper and auto-clicks close button on
- [IMDb Parents Guide Popup](https://greasyfork.org/en/scripts/568653-imdb-parents-guide-popup) - A user script that adds a popup with the IMDb Parents Guide
